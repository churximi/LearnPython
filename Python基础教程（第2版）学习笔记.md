#### **基本的列表操作**
- 元素赋值
- 删除元素：**del**
- 分片赋值

#### **列表方法**
- append （末尾追加新元素）
- count （统计元素出现次数）
- extend （追加列表，会改变被追加的列表）
- index （找出某个值**第一个匹配项**的位置）
- insert （插入元素，可分片）
- pop （移除元素，默认最后一个）
- remove （移出第一个匹配项）
- reverse （反向排序）
- sort （排序，不返回值）
- sorted （排序，返回值）
- 高级排序，如cmp()

#### **元组**（不可修改）
- tuple()  （相对于list()，其实是一种**类型**，而不是函数）
